# Basic Setup Guide

Welcome to the GenAI 2026 Programme! Follow these steps to get started.

---

## Step 1: Apply for GitHub Education Account

1. Go to [GitHub Education](https://education.github.com/students)
2. Click **"Sign up for Student Developer Pack"**
3. Sign in with your GitHub account (or create one)
4. Verify your student status using:
   - Your HKBU email address (`@life.hkbu.edu.hk`)
   - Or upload your student ID
5. Wait for approval (usually 1-7 days)

### Benefits You'll Get:
- GitHub Copilot (free for students)
- GitHub Codespaces
- Various developer tools and credits

---

## Step 2: Install Visual Studio Code

1. Download VS Code from [code.visualstudio.com](https://code.visualstudio.com/)
2. Install for your operating system (Windows/Mac/Linux)
3. Launch VS Code

---

## Step 3: Install Essential Extensions

Open VS Code and install these extensions:

1. **GitHub Copilot** - AI pair programmer
   - Press `Ctrl+Shift+X` (or `Cmd+Shift+X` on Mac)
   - Search for "GitHub Copilot"
   - Click Install

2. **GitHub Copilot Chat** - AI chat assistant
   - Search for "GitHub Copilot Chat"
   - Click Install

---

## Step 4: Sign In to GitHub

1. In VS Code, click the **Accounts** icon (bottom left)
2. Select **"Sign in with GitHub to use GitHub Copilot"**
3. Follow the browser prompts to authorize
4. Return to VS Code

---

## Step 5: Verify Setup

1. Create a new file (e.g., `test.py`)
2. Start typing: `def hello_world():`
3. GitHub Copilot should suggest code completions
4. Press `Tab` to accept a suggestion

### Test Copilot Chat:
1. Press `Ctrl+Shift+I` (or `Cmd+Shift+I` on Mac)
2. Type: "Explain what GitHub Copilot can do"
3. You should receive a response

---

## Troubleshooting

### Copilot not working?
- Ensure your GitHub Education account is approved
- Check you're signed in to the correct GitHub account
- Restart VS Code

### Need help?
- Open an issue in this repository
- Ask in the community discussions
- Contact your workshop facilitator

## Quick Reference

| Action | Shortcut |
|--------|----------|
| Open Copilot Chat | `Ctrl+Shift+I` / `Cmd+Shift+I` |

| See more suggestions | `Alt+]` / `Option+]` |
