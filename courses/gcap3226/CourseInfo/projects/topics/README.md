## Publishing with GitHub Pages

These HTML files are served from the **genAI2026** repo via GitHub Pages.

- **Base URL:** [https://tesolchina.github.io/genAI2026/](https://tesolchina.github.io/genAI2026/)
- **Settings:** Repo → **Settings** → **Pages** → Source: **Deploy from a branch** → Branch: **main** (or your default), folder **/ (root)**. Save. After a short delay, the links above will work.
- Ensure the repo is public and the branch containing these files is the one selected for Pages.
