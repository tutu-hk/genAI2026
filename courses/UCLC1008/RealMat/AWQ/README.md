# AWQ — Academic Writing Quiz materials

Materials are in **two layers** so you can find the most relevant ones quickly.

## Layer 1 → Layer 2

- **FromCourseTeam/**  
  - **Instructions** — Quiz instructions, pre-course, email from coordinator  
  - **Modules** — UE1 Module 1 & 2  
  - **RubricsAndCalendar** — Rubrics, academic calendar  

- **ExamTipsPractice/**  
  - **Samples** — Sample AWQ papers and student answers  
  - **StudyGuides** — Study guide to AWQ  
  - **MockPractice** — Practice tasks, word-count quiz  

- **ConceptSkillLesson/**  
  - **WeeklyPlans** — plan_week*, planAWQ, plan15, etc.  
  - **LessonContent** — week1–week5, week1.1–week5.3  
  - **MoodleTasks** — W3H1/W3H3 task HTML and READMEs  
  - **ReadingAndSynthesis** — Reading demos, synthesis/argument plans  
  - **Other** — READMEs, sample builds, InputProcessOutputModel  

Full mapping: **LAYER2_MAP.md**.  
To re-sort after a fresh copy from inventory: run **organize_layer2.py**.
