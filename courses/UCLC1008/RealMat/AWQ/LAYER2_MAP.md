# AWQ second-layer layout

Files and folders are grouped so the most relevant ones are easier to find.

## FromCourseTeam
- **Instructions** — Quiz instructions, pre-course, email from coordinator
- **Modules** — UE1 Module 1 & 2 (course content)
- **RubricsAndCalendar** — Rubrics, academic calendar

## ExamTipsPractice
- **Samples** — Sample AWQ papers and student answers
- **StudyGuides** — Study guide to AWQ
- **MockPractice** — Practice tasks, word-count quiz, set questions

## ConceptSkillLesson
- **skills4AWQ.md** — Main student reference: techniques and demo for reading, summarising, paraphrasing, synthesising, structure, citations, and prep
- **README.md** — Points to skills4AWQ.md and archive
- **archive/** — All original lesson materials (for reference or reuse):
  - *LessonContent* — week1–week5, week1.1–week5.3
  - *MoodleTasks* — W3H1/W3H3 task HTML and READMEs
  - *WeeklyPlans* — plan_week*, planAWQ, plan15, etc.
  - *ReadingAndSynthesis* — focusReading, taskOrientation, synthesisPlan, detailedPlan, etc.
  - *Other* — READMEs, sample_paraphrasing, InputProcessOutputModel
